System Overview

The PrimePath Assessment Platform is a two-step placement system that automatically assigns students to the appropriate curriculum level and exam based on their grade and English class rank.

How the Placement Process Works

1
Student enters their information: Grade level and English class rank (e.g., Primary 3, Top 10%)

4
Student takes the exam: The selected exam is presented to the student with timer and audio files

Key Components

  • Upload Exam: Upload PDF exams with naming convention [PlacementTest] followed by level and version (e.g., [PlacementTest] PRIME CORE PHONICS - Level 1_v_a)
  • Manage Exams: View, preview, and manage all uploaded exams and their audio files
  • Student Levels: Configure which curriculum level students are assigned to based on grade + English class rank
  • Level Exams: Map up to 5 exam versions to each curriculum level for random selection
  • Student Sessions: Monitor and track all student test sessions and results

Quick Setup Steps

  1. Upload exam PDFs using the Upload Exam tab (system will auto-name with version)
  2. Configure Student Levels to map student profiles to curriculum levels
  3. Set up Level Exams to assign exam versions to each level
  4. Share the student portal link for students to begin their placement tests
